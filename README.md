# Запуск
- **Production сборка**
	```bash
	# может понадобиться sudo
	npm install
	npm run build
	```
	
- **Сборка для разработки** ( в этом случае не будет минимализации файлов)
	```bash
	# может понадобиться sudo
	npm install
	npm run build:dev
	```
	После запуска этих команд в корне проекта появится директория **dist**, которую надо скопировать на сервер
	
# Используемые технологии
- **Компонентный VDom** с бэтчингом изменений состояний (`setState`)
- **JSX**
- **TypeScript**
- **Webpack**
- **WebSocket** - для совместного прослушивания
- **Broadcast Channel API** - для синхронизации между вкладками

# Возможности проекта

- Адаптация под все устройства
- Подборки музыки
- Музыкальный плеер
- Поиск (композиции, альбомы, исполнители)
- Страница исполнителя
- Страница альбома
- Избранное
- Пользовательские плейлисты (создание, просмотр, изменение, удаление)
- Шеринг
- Корректная работа при открытии нескольких вкладок
- Информация о статистике прослушивания
- История прослушивания
- Совместное прослушивание
- Интерфейс для работы с лейблами
